How to Run "Hello, World!"

This app loads a SWF into a browser window or an SGF (a converted SWF) onto a device.
  1. At the terminal "cd" into the "hello_world" directory.
  2. Launch a simple web-server to serve up the files. On Mac, you can just type the following line: `sudo python -m SimpleHTTPServer 80` That will serve the current directory up on port 80 of your computer.
  3. In a Web-Browser like Chrome or Firefox visit: "http://localhost/index.html?platform=flash"
  4. You should be able to play "Hello, World!" in your web-browser now using your Flash player. This is the standard environment for development and testing of Spaceport games.
  5. Now install the iOS build on a device to test it using the Spaceport iOS renderer
    1. Open up SpPublic.xcodeproj in the SpPublic folder using X-Code.
    2. Connect the device that you would like to test with (iPhone / iPad / iPod). Said device must already be provisioned with your team's provisioning profile so that you can code-sign.
    3. In X-Code select "device" and NOT "simulator" and pick the device that you just plugged in.
    4. Build and Run.
  6. Now that you have the "Spaceport App" installed on your iOS device, use it to open up "Hello, World!" that's running on your computer.
    1. Determine your internal IP address.
      1. At another terminal type: "ifconfig | grep inet"
      2. The last line should have something like: "inet 10.0.1.35 netmask 0xffffff00 broadcast 10.0.1.255" In this case, my internal IP is 10.0.1.35
      3. To verify that this is indeed the correct address, replace "localhost" with your IP in the previously used URL. For example, I would not test in my browser using the URL: "http://10.0.1.35/index.html?platform=flash"
    2. Make sure that your iOS device is connected to the same wifi network as your computer.
    3. Open the "Settings" menu on your iOS device. Find "Spaceport App" under "Apps". Fill in the "Server IP" field. For me, this would be: "10.0.1.35/index.html"
    4. Launch "Spaceport App". "Hello, World!" will now load and run.

Modifying the "Hello, World!" Code

Using a text editor, open up the "hello_world" directory. Make your changes and save them.

To view your changes in a browser, just select refresh and the updated game will now display.

To view your changes on an iOS device, first kill the currently running application. This is accomplished by double-pressing the home button to show a list of running applications, then touching and holding the "Spaceport App" icon in this list until it begins to shake. The app can then be terminated by touching the red circle. Launching the Spaceport App as indicated above will now display the updated game.

To debug, use either Firefox or Chrome. Firefox has an add-on called Firebug, and Chrome has "developer tools" under the "View > Developer" menu that will allow you set breakpoints, evaluate expressions, set watch expressions, inspect the call stack, etc.

Purpose of test is to determine if the color datatype is supported. There are multiple syntaxes for specifying the same color, such as #37F and #3377FF. For each of the nine groups shown here, each of the shapes in the group uses one of the syntactical forms and all in the group should be identical in color. This test is focussed on the X11 color names, which are not part of the tiny profile.

Each group uses four forms - 6-digit hex, rbg() integer form, rgb() percentage form, and named ('X11') colors. It does not use 3-digit hex, because the colors used in this test cannot be represented in three digit form.

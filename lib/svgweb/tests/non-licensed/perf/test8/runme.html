Flash Text Engine Tests

  • EmbeddedFont.html - Embeds a TrueType font directly into the Flash file to be rendered at runtime.
  • RuntimeFonts.html - Loads TrueType/OpenType English and Arabic fonts at runtime from separate files to style some text.
  • Bidi.html - Press the button to show bidirectional Japanese and some English in different directions.
  • MultiText.html - All the text from Gettysburg Address text demo styled with a TrueType font through Flash Text Engine to check performance.
  • webfont.html - Using CSS Web Fonts inside of HTML, including on Arabic text.
  • webfont.svg - Same, but using CSS Web Fonts inside of a *.svg file container, including on Arabic text.
